About Search Recipe App

In this section you can find more information about the "Recipe Search" application.

With Recipe Search finding recipes has never been easier. This application offers access to over 2.3 million recipes. Application was built with the goal to provide users with easy way to search for recipes. The application technology uses a well known and popular recipe database. The database contains number of different information about meals, nutritious values, ingredients and many more.
